# Bus Tracking Mobile App

React Native mobile app for real-time bus tracking.

## Features
- View available routes
- Track buses in real-time
- Interactive map
- ETA for stops
- Low bandwidth optimized

## Setup
```bash
npm install
# Update API_URL in src/config.js
npm start
```

## Run on Device
```bash
# Android
npm run android

# iOS
npm run ios
```

## Technologies
- React Native
- Expo
- React Native Maps
- Axios
- React Navigation
